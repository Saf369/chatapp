💬 ChatApp

A modern real-time messaging application that enables users to connect, chat, and share files seamlessly through an intuitive and responsive interface.

---

🚀 Features

- 💬 Real-time messaging
- 👥 One-to-one conversations
- 👨‍👩‍👧‍👦 Group chats
- 📁 File sharing
- 🔐 Secure authentication
- 🟢 Online/offline user status
- 📱 Responsive design for desktop and mobile devices
- 🌙 Dark mode support
- 🔒 End-to-end encryption
- ⚡ Fast and lightweight user experience

---

🌟 Overview

ChatApp is designed to provide a smooth and reliable communication experience. Whether you're chatting with friends, classmates, or colleagues, ChatApp delivers messages instantly while keeping conversations secure and easy to manage.
